# Design System: Portfolio - Rithwik Racharla
**Project ID:** 

## 1. Visual Theme & Atmosphere
**Job Winning, High Fidelity, Futuristic Professional**
A premium portfolio combining deep, immersive backgrounds with vibrant neon accents and glassmorphism. The aesthetic should feel "Agentic" and cutting-edge, utilizing smooth animations and interactive elements. It balances professional credibility with creative flair.
- **Mood:** Intelligent, Dynamic, Polished, "Next-Gen Tech".
- **Density:** Spacious, with breathable negative space.
- **Style:** Glassmorphism overlaying abstract 3D or gradient mesh backgrounds.

## 2. Color Palette & Roles
- **Deep Space Blue (Background):** A rich, dark void for depth (`#0B0E14`). Used for main backgrounds.
- **Neon Cyan (Accent/Primary):** Electric blue for calls to action and highlights (`#00F0FF`). Used for buttons, links, and key focus points.
- **Glass White (Surface):** Translucent white for cards and panels (`rgba(255, 255, 255, 0.05)` with blur). used for content containers.
- **Muted Lavender (Secondary Text):** Soft purple-grey for body text (`#A0A0B0`).
- **Pure White (Headings):** Crisp white for high-contrast titles (`#FFFFFF`).
- **Success Green:** Vibrant green for "Available for work" indicators (`#00FF9D`).

## 3. Typography Rules
- **Headings (Outfit/Inter):** Bold, modern sans-serif. Use `Outfit` for display (H1, H2) with tight tracking (-0.02em).
- **Body (Inter/Roboto):** Clean, legible `Inter` for long-form text. Light weights (300/400) for elegance.
- **Code/Monospace (Fira Code):** For technical terms or code snippets.

## 4. Component Stylings
- **Buttons (Primary):** Pill-shaped (`rounded-full`), Neon Cyan gradient background, subtle glow on hover (`box-shadow: 0 0 15px #00F0FF80`), text in dark blue/black.
- **Cards (Glass):** Generously rounded corners (`rounded-2xl`), frosted glass effect (`backdrop-filter: blur(12px)`), thin white border (`border: 1px solid rgba(255, 255, 255, 0.1)`).
- **Inputs:** Minimalist, underlined or pill-shaped with internal glass background.
- **Navigation:** Floating dock or sticky header with glass effect.

## 5. Layout Principles
- **Grid:** 12-column responsive grid.
- **Spacing:** Generous padding (80px+ between sections).
- **Alignment:** Center-focused for Hero, left-aligned for content.
- **Motion:** Everything should animate in (fade-up, slide-in). Hover states should be snappy and rewarding.
